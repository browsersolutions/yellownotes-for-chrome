Name
This column contains the name of the feed
The name can be updated by editing the cell directly. Any changes must be saved by clicking the save (floppy disk icon) icon at the end of the line.
Description
This column contains the description of the feed
The description can be updated by editing the cell directly. Any changes must be saved by clicking the save (floppy disk icon) icon at the end of the line.
Visibility
This column contains the visibility of the feed. A feed can be either "public" or "private".

PUBLIC The feed can be search for and subscribed to by any user.

PRIVATE The feed is only vilible to subscribers. The feed can only be subscribed to by invitation.
The visibility can be updated by selecting a new value from the drop-down list contained in the cell. Any changes must be saved by clicking the save (floppy disk icon) icon at the end of the line.
Restrictions
This column contains the restrictions that governs who can subscribe to the feed automatically. The restrictions can be either "NONE", "email" or "domain". Not yet implemented
post count
This column contains the number of YellowNotes attached this feed. You can sort the feeds by clicking ▲ (or ▼) in the column header.
not editable
subscriber count
This column contains the number of subscribers to this feed. You can sort the feeds by clicking ▲ (or ▼) in the column header.
not editable
Created
This column contains the date and time when the feed was created. You can sort the notes by creation date by clicking ▲ (or ▼) on the column header.
This information can not be modified
Active
This column contains the active/disabled status of the note. You can filter the notes by their status by selecting "active" or "inactive" from the dropdown menu.
Modify the notes active/inactive status by (un)ticking the checkbox. The change is immediate and there is not need to save the change.
anonymous_allowed
This indicated whether or not unauthentucated user may subscribe to this feed. Checkin this box means user needs not be logged in to subscribe to the feed. This is the default and is the recommended setting for sending YellowNotes to a large audience or new user.
Modify the permissions by (un)ticking the checkbox. The change is immediate and there is not need to save the change.
automatic_enrolment
This column indicates whether ot no user who satisfy the requiremetns are automatically approved. You can filter the feed by their status by selecting "active" or "inactive" from the dropdown menu.
Modify the permissions by (un)ticking the checkbox. The change is immediate and there is no need to save the change.
Actions
This column shows actions that can be take on the feed.

Delete. Deletion takes place immediately and there is no prompting for confirmation.

Notes. Opens the notes page for this feed.

Subscribers. Opens the subscribers page for this feed.

Save. Saves changes made to the text fields.

Invite. Create a invitation link for this feed that can be sent to other users.
